## 遊戲特色

- 即時戰鬥：操作簡單，但節奏緊湊
- 關卡推進：每關難度逐步提升
- Boss 對戰：需要走位與出招時機
- 手機優化：觸控操作，隨時開打

## 操作方式

- 左鍵：跳躍
- 右鍵：攻擊
- 長按攻擊：儲氣後釋放強力攻擊
